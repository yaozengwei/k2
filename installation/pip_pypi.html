Install using pip (pypi.org)

k2 on PyPI supports the following versions of Python, CUDA, and PyTorch:

Caution

We assume that you have installed cudatoolkit. If not, please install them before proceeding.

Hint

See also Install using conda (Linux/macOS). It installs all the dependencies for you automagically. You don’t need to pre-install PyTorch and cudatoolkit when using conda install.

The following command installs k2 from PyPI:

pip install k2

Caution

The wheel packages on PyPI are built using torch==1.7.1+cu101 on Ubuntu 18.04. If you are using other Linux systems or a different PyTorch version, the pre-built wheel packages may NOT work on your system, please consider one of the following alternatives to install k2:

  • Install using conda (Linux/macOS)

  • Install from source
